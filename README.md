# 🚀 Piyush Pandey - Streamlit Portfolio

A modern portfolio application built using Streamlit to showcase my skills, projects, internships, and experience in Data Analytics, Machine Learning, and Data Science.

## 🌟 About The Project

This portfolio application was developed to create an interactive and visually organized platform for presenting my technical background, projects, and professional experience.

The application highlights:
- Data Science and Analytics projects
- Internship experience
- Technical skills
- Educational background
- Interactive Streamlit-based UI

The goal of this project is to combine portfolio presentation with practical Streamlit development and deployment skills.

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Power BI

---

## 🚀 Featured Projects

### 🔹 Churn Rate Prediction Application
Machine learning application developed to predict customer churn using customer behavioral and demographic data.

### 🔹 HR Analytics Dashboard
Interactive Power BI dashboard for analyzing employee attrition trends and workforce insights.

### 🔹 Used Car Price Prediction
Regression-based machine learning application for predicting used car prices.
